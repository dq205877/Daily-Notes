```java
private static ListNode reverse(ListNode listNode){
        if(listNode == null){
            return null;
        }
        ListNode pre = null;
        ListNode cur = listNode;
        while(cur != null){
            ListNode next = cur.next;
            cur.next = pre;
            pre = cur;
            cur = next;
        }
        return pre;
    }
```
