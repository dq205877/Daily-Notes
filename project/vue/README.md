1.下载并安装nodejs(同时会有npm)

https://nodejs.org/zh-cn/download                  windows install

2.安装vue3

```
# 最新稳定版
$ npm init vue@latest
```

3.first vue project

```
npm install
npm run dev

# other cmd     or        browser open    localhost:5173
curl localhost:5173
```

4.cnpm

```
# 升级或安装 cnpm
$ npm install cnpm -g
$ cnpm -v
```
